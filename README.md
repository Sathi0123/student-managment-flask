# Student Management System (Flask + MongoDB)

This is a simple CRUD web application built using:

- Python Flask
- MongoDB
- HTML
- Bootstrap
- JavaScript Fetch API

## Features
- Add Student
- View Students
- Edit Student
- Delete Student
- Search Student

## How to Run

1. Install Python
2. Install MongoDB
3. Install dependencies

pip install flask pymongo

4. Run

python app.py

5. Open browser

http://127.0.0.1:5000
